# Thumbnails Creation Guide

## Requirements

Create **3 high-quality thumbnails** (1200×800px minimum) for the Framer marketplace listing.

### Thumbnail Specifications

- **Dimensions**: 1200×800px minimum (recommended: 1200×800px or 2400×1600px for retina)
- **Format**: PNG or JPEG
- **Quality**: High resolution, sharp, no pixelation
- **File Size**: Optimized but high quality (under 500KB recommended)

### Thumbnail Content

#### Thumbnail 1: Main Feature Showcase
**Focus**: Plugin UI with analysis results visible

**Content**:
- Show the plugin interface with Overview panel open
- Display page weight analysis (e.g., "2.4 MB")
- Show breakdown chart visible
- Highlight the clean, modern UI
- Use a real Framer project screenshot if possible
- Ensure text is readable at thumbnail size

**Design Tips**:
- Use a clean, uncluttered layout
- Highlight key metrics prominently
- Show both light and dark mode if possible (or choose one)
- Make sure the plugin UI is clearly visible

#### Thumbnail 2: Recommendations & Optimization
**Focus**: Smart recommendations and one-click optimization

**Content**:
- Show Recommendations panel with optimization suggestions
- Display "Top 3 Quick Wins" section
- Show "Resize & Compress" buttons visible
- Highlight potential savings (e.g., "Save 1.2 MB")
- Show before/after optimization comparison if possible

**Design Tips**:
- Emphasize the actionable recommendations
- Show the optimization workflow
- Highlight savings/impact clearly
- Use visual hierarchy to guide the eye

#### Thumbnail 3: Assets & Analysis Details
**Focus**: Asset management and detailed breakdown

**Content**:
- Show Assets panel with asset list
- Display file size badges and warnings
- Show asset previews/thumbnails
- Highlight the detailed breakdown view
- Show cross-page navigation capability

**Design Tips**:
- Show the comprehensive asset list
- Highlight file size indicators
- Display the organized, scannable layout
- Show the professional, polished interface

### Design Guidelines

**DO**:
- ✅ Show the actual plugin UI clearly
- ✅ Use high-quality screenshots
- ✅ Keep text minimal and readable
- ✅ Use Framer's design language
- ✅ Make it visually engaging
- ✅ Show real data/results
- ✅ Ensure sharp, clear images

**DON'T**:
- ❌ Add extra text labels or annotations
- ❌ Clutter the image with too much information
- ❌ Use low-resolution screenshots
- ❌ Add marketing copy or promotional text
- ❌ Use unauthorized Framer branding
- ❌ Show placeholder or fake data

### Creation Process

1. **Take Screenshots**:
   - Open the plugin in Framer Desktop
   - Navigate to each panel (Overview, Assets, Recommendations)
   - Take high-resolution screenshots (use Cmd+Shift+4 on Mac, or screenshot tool)
   - Ensure the plugin UI is clearly visible

2. **Edit & Optimize**:
   - Crop to 1200×800px aspect ratio
   - Ensure text is readable
   - Adjust brightness/contrast if needed
   - Remove any unnecessary UI elements if possible
   - Optimize file size while maintaining quality

3. **Save Files**:
   - Save as `thumbnails/thumbnail-1.png`
   - Save as `thumbnails/thumbnail-2.png`
   - Save as `thumbnails/thumbnail-3.png`
   - Verify dimensions are correct
   - Check file sizes are reasonable

### Tools

- **Screenshot**: Built-in OS tools or Snagit/CleanShot
- **Editing**: Figma, Photoshop, or any image editor
- **Optimization**: ImageOptim, TinyPNG, or similar
